# 🚀 פריסה ל-Render — רוני סטודיו כאתר עצמאי (נגיש מהטלפון)

המטרה: שהאפליקציה תרוץ בענן 24/7, ורוני תיכנס מהדפדפן בטלפון עם סיסמה — בלי מחשב דלוק, בלי Cowork.

## מה נפרס
- הדשבורד המלא (היום שלי · לידים · פניות · אולפן).
- מנוע ההפקה (Python+ffmpeg) מובנה ב-image — מוכן להפעלה מהשרת.
- נעילת סיסמה (Basic Auth) דרך `APP_PASSWORD`.
- אחסון קבוע לנתונים (Render Disk על `/app/data`).

## מפתחות שתצטרך (Environment Variables)
| מפתח | חובה? | למה |
|------|-------|-----|
| `APP_PASSWORD` | ✅ | הסיסמה שרוני מקלידה בכניסה |
| `PERPLEXITY_API_KEY` | אופציונלי | גילוי לידים חי עם מקורות (כפתור "מצא לידים") |
| `ANTHROPIC_API_KEY` | אופציונלי | ניסוח פניות/תסריטים אוטומטי (שלב הבא) |

## שלבים (פעם אחת)

**1. להעלות את הקוד ל-GitHub**
בתוך התיקייה `ronny-studio-app`, ב-Cursor → טרמינל:
```
git init && git add -A && git commit -m "ronny studio"
```
ואז ליצור repo חדש ב-github.com ולדחוף אליו (Cursor יציע את הפקודות, או: `git remote add origin <repo-url> && git push -u origin main`).

**2. חשבון Render** — להירשם ב-render.com (אפשר עם GitHub).

**3. ליצור את השירות** — New → **Blueprint** → לבחור את ה-repo. Render יקרא את `render.yaml` אוטומטית ויקים web service + disk.
(חלופה: New → Web Service → Docker → לבחור את ה-repo.)

**4. להגדיר Environment Variables** — להכניס `APP_PASSWORD` (סיסמה שתבחר), ואם רוצים: `PERPLEXITY_API_KEY`.

**5. Deploy** — Render בונה את ה-image (כמה דקות) ונותן כתובת כמו `https://ronny-studio.onrender.com`.

**6. לפתוח בטלפון** — הכתובת → דפדפן מבקש שם משתמש (כל דבר) + סיסמה (`APP_PASSWORD`). מוסיפים למסך הבית כאייקון.
