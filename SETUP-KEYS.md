# 🔑 חיבור לנתונים חיים — הוראות מפתחות

כדי שהאפליקציה תשאב מותגים מהאינטרנט **לבד ועם מקורות אמיתיים** (בלי להמציא), צריך מפתח API אחד.

## שלב 1 — מפתח Perplexity (גילוי לידים)
1. היכנס ל-https://www.perplexity.ai/settings/api
2. הוסף אמצעי תשלום (חיוב לפי שימוש — סנטים לחיפוש) וצור **API Key**.
3. העתק את המפתח.

## שלב 2 — הכנס את המפתח לפרויקט
1. בתיקייה `ronny-studio-app`, צור קובץ בשם `.env.local`.
2. הדבק בו:
   ```
   PERPLEXITY_API_KEY=המפתח-שלך-כאן
   ```
3. שמור, ועצור/הפעל מחדש את השרת (`Ctrl+C` ואז `npm run dev`).

## שלב 3 — שימוש
במסך **לידים** לוחצים על **"מצא לידים חדשים ✨"**, בוחרים נישה — והמערכת מוסיפה לידים אמיתיים עם לינק מקור לכל אחד.

> 🛡️ הגנה מהמצאות: כל ליד שאין לו לינק מקור אמיתי — נזרק אוטומטית ("cite-or-drop"). מה שנכנס — מגובה במקור.

## שלב הבא (אופציונלי)
`ANTHROPIC_API_KEY` — לניסוח פניות/תסריטים אוטומטי מתוך האפליקציה. נחבר כשנגיע לשלב 2–3 בתוכנית.
